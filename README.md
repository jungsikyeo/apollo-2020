# Apollo 2020

React, Apollo 및 GraphQL을 이용한 영화 웹 앱

```json
{
  "name":"영화 웹 앱",
  "skills": [
    "React"
    "GraphQL"
    "Apollo"
    "React Router"
  ]
}
```

## 상세 기능
- Replace Redux
- Replace Fetch
- Get Movies
- See Movie
- Add to Favourites
